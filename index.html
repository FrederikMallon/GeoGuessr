Daten & Sicherung

Alle Eingaben werden zuerst lokal in diesem Browser gespeichert (localStorage). Damit funktioniert das Tool sofort, auch ohne weitere Einrichtung — die Daten bleiben aber an dieses Gerät gebunden, bis du sie sicherst oder synchronisierst.

GitHub-Synchronisierung optional

Damit data/results.json im Repository selbst zum aktuellen Datenstand wird (z.B. um von mehreren Geräten aus einzutragen), kann das Tool die Datei direkt über die GitHub-API schreiben. Dafür wird ein fine-grained Personal Access Token mit Schreibrecht nur für dieses eine Repository benötigt. Der Token wird ausschließlich lokal in deinem Browser gespeichert und nur an api.github.com gesendet.
